============================================================
           AI JOB ALERT SYSTEM - PROJECT DETAILS
============================================================

1. PROJECT OVERVIEW
------------------------------------------------------------
The AI Job Alert System is an automated, full-stack application designed to aggregate freelance job postings from major platforms (LinkedIn, Upwork), intelligently score them using Google Gemini AI against a specific user profile, and deliver high-quality leads directly to the user via a modern web dashboard and email alerts.


2. TECHNOLOGY STACK
------------------------------------------------------------
* Backend:
  - Node.js & Express.js (REST API & Server)
  - better-sqlite3 (Lightweight Relational Database)
  - apify-client (For scraping Upwork securely)
  - @google/generative-ai (Gemini 2.5 Flash integration)
  - node-cron (Task Scheduling)
  - nodemailer (Email Alerts)

* Frontend:
  - React.js (Component-based UI)
  - Vite (Build Tool & Dev Server)
  - Tailwind CSS (Utility-first Styling)
  - Framer Motion (Animations & Transitions)
  - Lucide React (Iconography)


3. CORE ARCHITECTURE & WORKFLOW
------------------------------------------------------------
Phase 1: Aggregation (The Scrapers)
- LinkedIn: Scrapes real-time job listings using keyword matching.
- Upwork: Utilizes the official Apify Client (neatrat/upwork-job-scraper actor) to reliably bypass Cloudflare blocks and extract detailed Upwork listings.
- Fiverr: Removed from the pipeline to focus strictly on contract-based professional jobs.

Phase 2: AI Evaluation (Gemini 2.5 Flash)
- All newly scraped jobs are sent to Gemini 2.5 Flash.
- The AI compares the job description and budget against the user's "skills.json" profile.
- Output: An integer score (1-10), a list of matched skills, a 2-sentence personalized proposal snippet, and reasoning.

Phase 3: Smart Filtering (Noise Reduction)
- If a job scores less than 5 out of 10, the system automatically DELETES it from the database immediately to keep the feed clean.
- Only jobs scoring 5+ are saved to the dashboard.
- Jobs scoring 7+ trigger instant email alerts.

Phase 4: Dashboard Presentation
- The React frontend continuously polls the backend API to display the "Live Match Feed".
- Users can apply filters (e.g., "High Matches Only").
- Clicking "Apply Now" takes the user directly to the job listing equipped with their AI-generated proposal snippet.

5. HOW TO RUN THE PROJECT
------------------------------------------------------------
1. Open two terminal windows.
2. In Terminal 1 (Backend):
   > cd backend
   > npm install
   > node src/index.js
   (The backend runs on port 3001 and triggers the cron job every 15 minutes)

3. In Terminal 2 (Frontend):
   > cd frontend
   > npm install
   > npm run dev
   (The frontend runs on port 5173 and proxies API requests to port 3001)

4. Manual Scrape Trigger:
   Send a POST request to http://localhost:3001/api/run to bypass the 15-minute timer and force an immediate scrape cycle.
